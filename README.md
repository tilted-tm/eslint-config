# Tilted eslint config

## Installation

```shell script
yarn add @tilted/eslint-config --dev

# or using npm

npm install @tilted/eslint-config --save-dev
```

## Usage

### Simple js
Using `.eslintrc.json`:
```json
{
  "extends": "@tilted"
}
```

Using `.eslintrc.js`:
```javascript
module.exports = {
  extends: '@tilted'
}
```

### React
Using `.eslintrc.json`:
```json
{
  "extends": "@tilted/react"
}
```

Using `.eslintrc.js`:
```javascript
module.exports = {
  extends: '@tilted/react'
}
```
